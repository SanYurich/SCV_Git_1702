# Работа с Git

## 1. Проверка наличия установленного Git

В терминале выполнить команду `git version`
Если Git установлен, появляется сообщение с информацией о версии программы. Иначе будет сообщение об ошибке.

## 2. Установка Git
Загружаем последнюю версию Git с сайта https://git-scm.com/downloads

## 3. Настройка Git
При пераом использовании Git необходимо представиться. Для этого нужно ввести в терминале две команды:
```
git config --global user.name "Ваше ИМя"
git config --global user.email "email@example.com"
```
